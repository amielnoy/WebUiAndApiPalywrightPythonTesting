## Instalation(for mac/linux):

1. Install Python 3.12
2. install pip
3. Run(install virtual env)
   python3 -m venv .venv
   source .venv/bin/activate
4. pip install -r requirements.txt

## Run Tests:
pytest

# Run Tests in parallel:
pytest -n auto
